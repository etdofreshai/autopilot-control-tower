# Task Taxonomy

- Epic: multi-wave outcome, multiple stories.
- Story: one model owns a coherent feature slice end-to-end.
- Task: small support work.
- Experiment: speculative improvement with rollback/eval.
- Review: critique or verification only.
- Rescue: fix failed/conflicted work.

Default target: story-sized work, roughly 10x larger than current micro-tasks, with explicit acceptance checks.
